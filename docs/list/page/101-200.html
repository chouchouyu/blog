200,"agnostic", n 不可知论者（无法证实上帝的存在但又不否认上帝存在的人）；adj 不可知论的

【源】不可知论者并不否认上帝和天堂的存在，但却认为无法确切知道他们的存在与否。agnostic这个词是由19世纪的英国科学家赫胥黎创造出来的。他用前缀 a-意为“没有，无”和名词 Gnostic造词。Gnostic与希腊词 gnosis有关，指“意识：关于精神玄奥的、神秘的意识”。 因此 Gnostic就指那些有这类意识的人。在创造 agnostic这个词时，赫胥黎把他的同伴认为是“可知论者”。

【参】diagnostic（adj 诊断的）；ignorance（n 愚昧）"
