# 1.字符串的解构赋值

对象形式的解构赋值

const { 0: a, 1: b, length } = 'hello';

console.log('hello'.length);

//由于数组本质是特殊的对象，因此可以对数组进行对象属性的解构。
let arr = [1, 2, 3];
let {0 : first, [arr.length - 1] : last} = arr;
console.log(first); //1
console.log(last); //3

数组/对象解构赋值的原理

注意:数组是索引值，对象是属性名
